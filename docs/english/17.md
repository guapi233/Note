# 短语短句

## 被动语态

```
主动
The terrorists bomed the station.

被动
The station was bombed by the .
```

be和过去式v是不会结合在一起的，所以be后面只可能跟的是p.p.

**主动语态变被动语态**

1. 将主动语态中的宾语变为被动语态中的主语。

2. 把谓语动词变成被动语态，时态要和主动语态的时态一致。

3. 主动语态的主语变为介词by的宾语，人称代词主格要变为宾格。

**There be 句型中的不定式用主动表被动意义。**

```
There is much work to finish
有许多工作要完成。 （工作是被人完成的）
```

**too…to…结构中的不定式用主动表被动意义。**

```
The box is too heavy to carry.
这个箱子太重而不能搬动。（道理同上）
```



**不能用被动语态的情况**

1. 谓语动词为不及物动词或不及物动词短语时，不用被动语态。如：happen，take place，take off等。

2. 有些及物动词或短语也无被动语态。如：cost，leave，enter，reach，have，wish，belong to，mind等。

3. 宾语为反身代词、相互代词、不定式、现在分词等时，不能变为被动语态。

5. 系动词无被动语态。如：`be,become,feel,get,look,seem,smell,sound,taste,turn`等。



当不知道或没有必要指出动作的执行者时，常用被动语态，这时往往不用by短语。

```
The front window in the classroom was broken yesterday. 昨天，教室的前窗被打破了。(不知谁打破的)

They have been poorly paid. 他们的工资太低。(没必要指出工资是谁付的)
```



突出或强调动作的承受者，如果需要说出动作的执行者，用by短语。

```
The time-table has been changed. 时间表已变动了。(要突出的是“时间”)

These books are written especially for children. 这些书是专门为孩子们写的。(强调的是“”这些书)
```



为了使语言得体或圆滑等不愿意说出动作的执行者。如：

```
You are requested to make a speech at next meeting. 请您在下次会议上作个发言。

It is said that she is going to be married to a foreigner. 据说她要嫁给一个外国人。
```



出于修辞的原因，或是说为了更好地安排句子。如：

```
It is generally considered impolite to ask one’s age, salary, marriage, etc. . 问别人的年龄、工资、婚姻状况等通常被认为是不礼貌的。

The notion is especially opposed to by those who have invested much in the area. 这个提议特别遭到了那些在本地区投资很大的人的反对。(因those的定语太长了，若用它作主语，主语与谓语就相距太远而显得句子松散。)
```



## can't have

表示“不允许”

```
I can't have the children playing in my room.
我不允许小朋友在我的房间里玩。
```



## have + O. + 现在分词

表“使，让”

```
She had the children laughing again.
她使小朋友再一次笑起来。（表示笑的一个状态）
She had the children laugh.
她让小朋友笑。    （表示一种要求，笑没笑也不知道）
```



## have + O. + 过去分词

表“使得，使役”

```
I had my hair cut yesterday.
（我使得我的头发被剪）我理发了。
```

表”被害“

```
I had my money stolen.
我的钱被偷了
```



## 分词构句

由分词为首的句型。表示时间、条件、理由或目的等。



```
When I opened the door, I heard a strange sound.
（表示时间）				（主要子句）

Opening the door, I heard a strange sound.
（句意不变的情况下，可省略连接词）
```

being后面跟形容词或名词的情况下，being不可以被省略。



**表条件**

```
If turn left after the bank, you will see our house on the right.
```



**独立分词构句**

```
因为星期天，所有的商店都打烊。
Because it was Sunday, the stores were all closed =>
It being Sunday, the stores were all closed.
```
