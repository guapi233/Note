# 第一章

## 什么是编译器？

* 计算设备包括个人计算机、大型机、嵌入式系统、智能设备等
* 核心的问题都是软件的构造
  * 而目前绝大部分软件都是由高级语言书写
  * 成百种高级语言
* 这些语言是如何运行在计算机系统上的？
* 编译器是一个**程序**
* 核心功能是把**源代码**翻译成**目标代码**

 

## 编译器和解释器

* 编译器：源代码  ==>  编译  ==>  可执行程序
* 解释器：源代码  ==>  编译  ==>  执行结果
